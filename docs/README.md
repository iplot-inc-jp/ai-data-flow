## プロジェクト構成

```
ai_data_flow/
├── docs/                    # 設計書（このディレクトリ）
├── frontend/                # Next.js フロントエンド
├── backend/                 # NestJS バックエンド（Docker）
└── shared/                  # 共有型定義
```

## 技術スタック概要

- **フロントエンド**: Next.js 14 (App Router), React Flow, Tailwind CSS, shadcn/ui
- **バックエンド**: NestJS, Prisma, PostgreSQL
- **インフラ**: Docker, Docker Compose
- **認証**: NextAuth.js + JWT

## 用語集

| 用語 | 説明 |
|------|------|
| データカタログ | テーブル・カラムのメタデータを管理するシステム |
| 業務フロー | 業務プロセスを可視化したフローチャート |
| ロール | 業務を担当する主体（人/システム/その他） |
| CRUDマッピング | どのロールがいつどのデータを操作するかの定義 |
| IF定義 | ロール間のデータ受け渡しインターフェース定義 |
| スイムレーン | ロール別に区切られたフロー図のレーン |
